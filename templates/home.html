Learn programming in 4 short courses.

Course 0: An overview.

In this course, you'll learn what programming is. We'll look at the anatomy of four different applications, and you'll learn what different skills are required for different types of application.

Course 1: Rudy the Red Dot & JavaScript Syntax

In this course, you'll learn the syntax of the JavaScript programming language. JavaScript is a required language for web applications, but it's not often used for other types of programs. However, the syntax of the JavaScript language is quite similar in many respects to Objective-C (used for iPhone/iPad apps), Java (used for Android apps), and C++ (used for desktop apps). Once you know JavaScript, it will be much, much easier to learn other languages.

Course 2: Processing & Principles of Software Engineering

This course introduces Processing, a software library that lets you produce interactive art sketches. In this course, you'll learn about how to use a software library, how to break a hard problem down into smaller, easier parts, and how to work with user input.

Sidebar: UNIX and the command line.

Knowing the basics of the UNIX command line is a requirement for professional programming.
